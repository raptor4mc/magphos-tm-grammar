# MagPhos TextMate Grammar

This repository contains the official TextMate grammar for the **MagPhos** programming language.

It provides syntax highlighting for:

- `.mp`
- `.mgph`
- `.magphos`

Used by GitHub Linguist and editors that support TextMate grammars.

## Features
- Keywords: `var`, `set`, `if`, `else`, `loop`, `print`
- Strings
- Numbers
- Operators
- Braces
